# 📊 Fundamental Booster: Data Analytics Suite

Welcome to the **Fundamental Booster** project. This repository contains a comprehensive collection of datasets designed to demonstrate core data processing, analytical modeling, and business intelligence techniques across multiple domains including Education, Sales, and Human Resources.

---

## 📂 Dataset Overview

The project is structured into three primary analytical modules, each focusing on specific Key Performance Indicators (KPIs):

### 1. 🎓 Student Grade Analytics

Focuses on academic performance tracking and grade distribution.

* **Core Data:** Tracks Student IDs, Names, and individual scores in Math, Science, and English.
* **Computed Metrics:** Includes automated calculations for Total Scores, Averages, and assigned Letter Grades.
* **Advanced Features:** Utilizes enrollment timestamps to analyze performance trends over time.

### 2. 📈 Sales & Revenue Intelligence

A transactional dataset designed for retail and commercial analysis.

* **Tracking:** Monitors Sales IDs, Product types, and Regional performance.
* **Personnel Management:** Links specific Salespeople to transaction amounts.
* **Strategic Logic:** Implements tiered discount logic (e.g., 10%, 20%) and lookup functions for revenue optimization.

### 3. 👥 Employee HR Management

Streamlines workforce data and compensation structures.

* **Organizational Structure:** Categorizes employees by Department (Finance, IT, HR, Marketing).
* **Compensation Tracking:** Maintains detailed records of Salaries and specific joining dates.
* **Time-Series Analysis:** Includes extracted joining years and months for tenure and retention modeling.

---

## 🛠️ Technical Implementation

This suite serves as a benchmark for implementing the following data operations:

| Feature | Implementation Highlights |
| --- | --- |
| **Lookup Logic** | Integrated `VLOOKUP` and `XLOOKUP` demonstrations for cross-referencing IDs. |
| **Conditional Logic** | `COUNTIFS` and `FILTER` functions applied to academic and sales data. |
| **Data Cleaning** | Standardization of dates and numerical formatting for financial reporting. |
| **Statistical Analysis** | Calculation of averages and weighted scores across various categories. |
